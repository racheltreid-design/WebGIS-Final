# US Cities
 
## Description
This web map shows three major cities in the US and some information about them.
It was built using Leaflet.js as part of GIS 5220/420 Lab 6.
 
## Data Source
The data used in this map comes from the assignment example. (Dr. Ghosh if you're reading this I am going to do another lab with my own example)
 
## Features
- Interactive basemap with zoom and pan
- GeoJSON point markers for US cities
-  Clickable popups showing city name and population
- Scale bar showing metric and imperial units
 
## Technologies Used
- HTML5, CSS3, JavaScript (all in one index.html file)
- Leaflet.js 1.9.4
- OpenStreetMap tiles
